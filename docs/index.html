MeshCore Packet Knife

#
Brute Force

Search for the room name by trying all valid combinations. Only works for #channel rooms (not private keys), and you can't know if a packet was a GroupMessage to a hashtag room or otherwise.

Cracking performance is affected by browser throttling when backgrounded; keep the window in the foreground.

Assumes room names start with #, use only a-z, 0-9, and dash. Dashes cannot be leading, trailing, or consecutive.
