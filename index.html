#
Find Room Name

Search for the room name by trying all valid combinations. Only works for #channel rooms (not private rooms with random keys), and you can't know if a packet was a GroupText to a hashtag room or otherwise. Read more about how this works in the README.

Assumes room names start with #, use only a-z, 0-9, and dash. Dashes cannot be leading, trailing, or consecutive.

Search performance is affected by browser throttling when backgrounded; keep the window in the foreground.

MeshCore is still secure! Nothing is hacked or broken about MeshCore due to this demo. Hashtag rooms are not considered private; this merely demonstrates that reality. MeshCore has strong, modern, secure crypto for private messages and private rooms. Use them, and not hashtag rooms, for private communications, and you're just fine.
